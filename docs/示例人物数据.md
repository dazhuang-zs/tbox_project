生平故事

summary: 李白（701年-762年），字太白，号青莲居士，又号谪仙人，唐代伟大的浪漫主义诗人，被后人誉为诗仙。与杜甫并称李杜，其诗风格豪放飘逸，想象丰富，语言流转自然，音律和谐多变，善于从民歌、神话中汲取营养，创造出瑰玮绚烂的色彩。

detailed: 李白出生于西域碎叶城，五岁随家迁居四川绵州昌隆县。自幼聪慧好学，博览群书，尤其喜好道家思想。十五岁开始游历名山大川，结交天下豪杰。二十五岁时仗剑去国，辞亲远游，足迹遍及大半个中国。天宝元年，经道士吴筠推荐，被唐玄宗召入长安，供奉翰林。但因性格狂放不羁，得罪权贵，三年后被赐金放还。此后继续漫游天下，广交天下英才。安史之乱期间，因参与永王李璘幕府而受牵连，被流放夜郎，途中遇赦。晚年投奔族叔李阳冰，最终病逝于安徽当涂，终年六十二岁。
