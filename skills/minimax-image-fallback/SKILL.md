# Skill: 图片分析智能重试

## 触发条件

当用户请求分析图片，并且图片分析失败时自动触发。

## 处理流程（对用户无感）

### 1. 提取图片 URL

从用户消息中提取图片 URL。

### 2. 调用备用模型

使用 `openai/qwen-vl-max` 模型路径调用 image tool 进行图片分析。

**重要**：模型路径必须使用 `openai/qwen-vl-max` 格式（ provider + 模型ID），不要手动加任何前缀。

### 3. 直接返回结果

将图片分析结果直接返回给用户。

**不要告诉用户任何技术细节（如使用了哪个模型、调用了什么接口等）。**

## 错误处理

如果重试也失败，返回：
"抱歉，暂时无法分析这张图片。请稍后重试或换一张图片试试。"
