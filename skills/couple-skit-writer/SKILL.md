# 情侣短视频剧本创作技能

专为抖音/快手/小红书等平台创作高质量情侣短视频剧本。核心目标：**让演出来的段子，看起来像真的。**

## 工作流程

1. 读取详细方法论 `references/full-guide.md`
2. 确认用户需求：平台、时长、风格、人设
3. 按标准格式输出完整分镜脚本

详细创作方法论：
```
references/full-guide.md          # 完整写作方法论（核心文件）
references/hook-library.md        # 100个开篇钩子库
references/conflict-patterns.md   # 30个冲突场景模板
references/character-design.md    # 人设深度设计指南
references/trend-hijacking.md     # 追热点方法论
references/anti-script-trace.md   # 去编剧痕迹15条铁律
```

## 输出格式

```
标题：[平台适配标题]
时长：[X]秒
风格：[甜宠/搞笑/扎心/治愈/反转]
热点关联：[如有]

人物：
- 女生：[名字]，[3个性格标签]，[1个反差萌点]
- 男生：[名字]，[3个性格标签]，[1个反差萌点]

核心冲突：[一句话]

完整分镜脚本：

| 镜号 | 时长 | 画面 | 台词 | 音效/BGM | 表演指导 |
|------|------|------|------|----------|----------|
| 1 | 2s | ... | ... | ... | ... |

创作说明：
- 钩子类型：
- 情感落点：
- 爆点位置：
- 预期互动类型：
```
