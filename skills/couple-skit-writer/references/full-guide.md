# 情侣短视频剧本创作完整方法论

## 第一部分：行业底层认知

### 1.1 观众为什么爱看情侣段子？

不是因为想看别人谈恋爱。是因为**在里面看到了自己**。

三个心理机制：
1. **投射**：看到女主吃醋，想起自己上次也是这么作的——"这不就是我吗"
2. **补偿**：现实中男朋友不浪漫，看别人甜甜的恋爱过过瘾——"我什么时候能有这种男朋友"
3. **社交货币**：转发给男朋友说"你看看人家"——评论区艾特对象，就是最高级的传播

所以写剧本的第一原则永远是：**写的是情侣，戳的是人性。**

### 1.2 平台差异化（同一个本子怎么适配三个平台）

| 维度 | 抖音 | 快手 | 小红书 |
|------|------|------|--------|
| 时长 | 30-60秒最佳 | 15-30秒最佳 | 45-90秒最佳 |
| 节奏 | 快，每5秒一个信息点 | 极快，3秒一爆点 | 中速，可以有一些留白 |
| 音乐 | 热门卡点BGM，踩节奏 | 接地气音效+方言 | 氛围感轻音乐 |
| 人设 | 颜值+反差 | 真实+接地气 | 精致+生活感 |
| 冲突浓度 | 中高 | 高 | 中 |
| 甜度 | 中高 | 中 | 高 |
| 评论区风格 | 搞笑段子手聚集 | 老铁真实分享 | 精致女孩互夸 |

## 第二部分：人设打造（剧本的灵魂）

### 2.1 人设黄金公式

```
人设吸引力 = 标签化（1秒被记住）+ 反差萌（3秒被喜欢）+ 成长弧（一直追下去）
```

**标签化（必须）：**

一个人设应该在观众看到的第一眼就被理解。用三标签法：

```
她是一个 [职业/身份] + [性格标签] + [独特癖好]
他是一個 [职业/身份] + [性格标签] + [独特癖好]
```

示例：
- 她：大厂产品经理，表面雷厉风行，私下怕蟑螂会跳到椅子上尖叫
- 他：自由摄影师，看起来文艺高冷，实际上每天给她发20条"在干嘛"
- 她：幼师，对小朋友耐心爆棚，对男朋友三秒就炸
- 他：外科医生，手术台上稳如泰山，女朋友一哭就手忙脚乱

**反差萌（必须）：**

反差萌的设计原则：**强大的人设 + 软肋 = 让人心里一软**

| 人设方向 | 反差萌设计 |
|----------|------------|
| 高冷型 | 私下是个黏人精 / 偷偷给女朋友写小作文 |
| 强势型 | 女朋友一撒娇就投降 / 在外面霸气回家秒怂 |
| 搞笑型 | 认真的时候最帅 / 其实比谁都细腻 |
| 温柔型 | 吃醋的时候判若两人 / 有不可触碰的底线 |
| 直男型 | 笨拙但真诚的浪漫 / 偷偷做了功课 |

**成长弧（进阶）：**

真正的长期账号，人设要有变化。不是一天一个样，而是"追了你30条视频，发现她慢慢变了"。
- 从"天天查手机"到"我信你"
- 从"只会说多喝热水"到"记得你生理期比你还清楚"
- 从"动不动就冷战"到"生气归生气，但舍不得让你难过"

### 2.2 人设禁忌（踩了必死）

- 人设不自洽：前一天还在撒狗粮，后一天就苦大仇深。除非是"系列反转"，否则不要这么搞
- 人设标签堆积超过5个：观众记不住
- 人设过于完美：没缺点的人设=没有人味
- 人设和演员本人差距过大：演不出来，尬

## 第三部分：冲突设计（剧本的发动机）

### 3.1 冲突的黄金三要素

每一次有效的冲突，都必须同时满足：

```
信息差 + 情绪差 + 利益差
```

**信息差（观众知道，角色不知道）：**
- 观众知道男主买的是戒指，女主以为他出轨了 → 全程揪心
- 观众知道女生在准备惊喜，男生以为她最近不爱自己了 → 全程心疼

**情绪差（两个角色的情绪不在同一频道）：**
- 她很认真地在吃醋，他以为她在开玩笑 → 情绪错位制造笑点
- 他鼓起勇气说爱，她在刷手机完全没听 → 反差制造共鸣

**利益差（两个角色想要的不一样）：**
- 她想出去约会，他想在家打游戏
- 她想公开恋情，他觉得现在不是时候

### 3.2 冲突浓度梯度（从轻到重）

| 级别 | 类型 | 示例 | 适用场景 |
|------|------|------|----------|
| Lv.1 撒娇级 | 小事闹一闹 | "你居然没发现我换发色了" | 甜宠日常 |
| Lv.2 吃醋级 | 第三方出现 | "那个女同事又给你发消息了" | 搞笑段子 |
| Lv.3 误会级 | 信息不对称 | "我明明看到了你还说没有" | 反转剧 |
| Lv.4 原则级 | 触及底线 | "你说过戒烟的你又在抽" | 扎心治愈 |
| Lv.5 信任级 | 关系危机 | "你到底还爱不爱我" | 情绪高潮 |

一条视频里，冲突浓度应该走一个曲线：**Lv.1→Lv.3→Lv.5→和解**，或者**Lv.3→Lv.1→反转冒甜**。

不要在一条视频里从Lv.5起步——观众还没代入就崩了。先从小事铺垫，让观众"认识"这对情侣，再升级冲突。

### 3.3 30秒内完成冲突循环的极简模板

```
0-3s：冲突爆发（Lv.2-3）
  画面：她拿着手机，脸色变了
  台词："你是不是又给那个XXX点赞了？"

4-10s：情绪升级（Lv.3-4）
  画面：他试图解释，她越听越气
  台词："你每次都这样说！""那你想让我怎么说？"

11-20s：冲突高潮（Lv.4-5）
  画面：两人对峙，背景音紧张
  台词：她说了一句最狠的话，空气凝固

21-27s：转折/反转
  画面：一个意外信息打破了僵局
  关键：转折必须有铺垫（前5秒埋的线索）

28-30s：收尾
  画面：她破涕为笑 / 他松了口气 / 两人对视
  BGM：从紧张切到温暖
```

## 第四部分：对话写作（"像真的"的核心）

### 4.1 对话去水14条铁律

1. **每句不超过15字。** 超过就拆。短视频不是演讲，没人听你长篇大论。

2. **删掉所有"你好""请问""我觉得"开头。** 直接说事。不是"我觉得你最近有点奇怪"→ 直接"你最近很奇怪。"

3. **说半句，让对方接。** 真实对话里没有完整的句子。不是"你是不是不爱我了"→ 而是"你是不是……"然后沉默。

4. **用语气词替代形容词。** "很生气"→ "哼""哦""行吧"。语气词比形容词有画面感。

5. **每三句对话必须推进剧情。** 两个人的对话不能原地踏步。要么信息增多了，要么情绪升级了，要么事情发生了。原地打转的对话直接删。

6. **吵架时用短句（5字内），甜蜜时可以稍微长一点（15字内）。** 吵架语速快、字少、打断多。甜蜜时徐徐展开。

7. **不要在对话里解释背景。** 最差的台词长这样："你还记得我们上次去那个餐厅吗就是上周五"——这是给观众读的说明书。正确的做法是："上次那家——""别提了。"

8. **让角色说错话然后纠正。** 真实感的关键之一。不是"我想说的是，我觉得我们需要谈谈"→ 而是"我想说……算了我不说了……不对我得说。"

9. **用动作打断对话。** 不是每句话都要回。"你到底爱不爱我！"——他不说话，走过去抱住她。比任何台词都有力量。

10. **方言和口癖是最好的记忆锚点。** 一句"你咋又这样咧"比一万句标准普通话有辨识度。每个人可以设计1-2个标志性口癖。

11. **对话戛然而止比圆满结尾高级。** 最高级的结尾是观众自己脑补的那种。她说"好吧，原谅你了"，切黑屏——观众会自己填剩下的内容。

12. **反问比陈述有力三倍。**

13. **每场戏最多两个人同时说话。** 情侣视频就两个主角，偶尔加一个闺蜜/哥们。不要搞群戏。

14. **观众的注意力不是平均分配的，是脉冲式的。** 对话中间每隔5-8秒需要有一个"注意力锚点"——一个重音词、一个停顿、一个眼神、一个音效。

### 4.2 五种对话风格的配方

| 风格 | 特征 | 节奏 | 示例 |
|------|------|------|------|
| 甜宠风 | 短句+语气词+眼神 | 慢，留白多 | "你看着我。""嗯。""没什么，就是想看。" |
| 搞笑风 | 快接+错位理解+callback | 快，停不下来 | "我想要星星。""楼下超市有杨桃，长得差不多。" |
| 扎心风 | 长句开头→短句收刀 | 先慢后快 | "你知道吗，我最难过的是……你根本没发现。" |
| 治愈风 | 重复+递进+安静 | 极慢，配纯音乐 | "没事的。""嗯。""真的没事的。""我知道。" |
| 反转风 | 误导→真相爆破 | 快→停→更快 | 前半句把人带入误区，后半句一锤定音 |

## 第五部分：节奏控制（完播率的密码）

### 5.1 短视频的黄金时间线

```
00:00-00:03  钩子区：决定60%的观众留不留
00:03-00:08  代入区：让观众"对号入座"
00:08-00:15  升级区：矛盾升级，吸引看下去
00:15-00:25  高潮区：最精彩的部分
00:25-00:35  反转区：出人意料
00:35-00:45  落点区：情绪收束+互动引导
00:45-00:60  余味区（可选）：一个callback让观众倒回去重看
```

### 5.2 每5秒必须发生一件事

检查你的剧本：每5秒，观众看到了什么新的东西？如果没有——砍掉这5秒。

"新东西"包括：
- 剧情推进（一个动作、一句关键台词）
- 情绪变化（从开心到疑惑、从难过到愤怒）
- 画面变化（场景切换、镜头移动、重要道具出现）
- 信息揭露（一个反转、一个真相、一个暗示）

### 5.3 BGM使用的三条铁律

1. **不要在台词密集的时候铺带歌词的BGM。** 歌词和台词互相干扰。台词段用纯音乐或轻BGM，留白段才上歌词。

2. **BGM的切换就是情绪的切换。** 冲突爆发→BGM骤停，反转真相→舒缓进，甜蜜结尾→轻快起。BGM切换点要和情绪转折精确对齐。

3. **用热门BGM不要只在开头放一下。** 要么不放，要么全段跟着BGM的节奏踩点。半吊子的BGM使用比不用更毁节奏。

## 第六部分：追热点方法论

### 6.1 热点分层

| 层级 | 类型 | 时效 | 举例 |
|------|------|------|------|
| S级 | 全民热点 | 24-48h | 重大新闻、全民讨论的社会事件 |
| A级 | 平台热点 | 3-7天 | 抖音挑战赛、平台热门话题 |
| B级 | 圈层热点 | 1-2周 | 某个梗、某首歌、某部剧 |
| C级 | 情感热点 | 长期 | 情人节、520、七夕、毕业季 |

### 6.2 热点嫁接模板

```
热点事件 + 情侣视角 = 爆款

模板：
"全网都在讨论[热点]，那如果是你女朋友，她会……"

示例：
- "全网都在说GPT-6多厉害，那你猜我女朋友用它干嘛——写情书，而且是AI写给我的那种"
- "大家都在聊DeepSeek V4有多便宜，我男朋友只关心一件事——能不能拿来算双十一满减"
```

嫁接原则：
- 热点太严肃→用情侣视角软化（"你看这个新闻"→"你看他看新闻的时候我又在吃醋"）
- 热点太娱乐→用情侣视角加真实感（"他们好甜"→"你上次也这样说，然后呢？"）
- 热点太垂直→不追，强行追等于自嗨

### 6.3 永不失效的"日历热点"

| 时间 | 热点 | 创作方向 |
|------|------|----------|
| 1月 | 跨年、年终 | 跨年第一条消息发给谁 / 年终总结里有没有她 |
| 2月 | 春节、情人节 | 回谁家过年 / 直男情人节礼物翻车 |
| 3-4月 | 春天、樱花 | 春天的第一场约会 / 他偷偷拍的她 |
| 5月 | 520、毕业季 | 表白策划 / 毕业不分手 |
| 6月 | 高考、夏天 | 那些年我们一起…… / 她高考时他在门外 |
| 7-8月 | 暑假、七夕 | 异地恋 / 七夕礼物大翻车 |
| 9月 | 开学、秋天 | 新学期新室友撮合 / 秋天的第一杯奶茶 |
| 10月 | 国庆、中秋 | 长假旅游吵架 / 月亮代表我的心 |
| 11月 | 双十一 | 购物车清空 / 帮她凑满减 |
| 12月 | 圣诞、跨年 | 圣诞礼物 / 年末表白 |

## 第七部分：个人特色打造

### 7.1 找到你的"一句话辨识度"

问自己：你的账号和其他情侣号唯一的区别是什么？

- "全抖音最直男的男朋友"——这句话让人立刻有画面
- "每次吵架都是我哄她，因为我有把柄在她手里"——有悬念
- "我们是相亲认识的，但她到现在都不知道我是她的甲方"——有故事

人设标签越窄，粉丝粘性越强。不要做"甜宠情侣"，要做"被女朋友欺负但很享受的程序员男友"。

### 7.2 三个让账号有辨识度的技巧

**技巧一：固定一个"只有你们会这样"的梗**

每条视频都出现的元素，让粉丝一看就知道是你们。
- 一个固定的转场方式（每次吵架结尾都是他默默去厨房做饭）
- 一句固定的开场白或口头禅
- 一个固定的场景（沙发、厨房、车里）

**技巧二：设计一个"粉丝专属梗"**

让粉丝觉得自己不是观众，是"自己人"。
- 一个只有老粉才懂的昵称（"那个穿蓝色卫衣的男人"）
- 一个固定的评论互动方式（"今天他又干了什么蠢事，评论区告诉我"）

**技巧三：敢于暴露"不完美"**

完美情侣号=没人信。最火的情侣号都是：
- 她素颜出镜，他头发乱糟糟
- 吵架也很真实，不是那种"演"出来的吵架
- 偶尔也丧，不是每天在撒糖

## 第八部分：实战案例拆解

### 案例一：经典吃醋段子还原

**成功案例的底层逻辑：**

"你为什么点赞她？"
这六个字是短视频史上最成功的吃醋开场白。为什么？

1. 人人都有共鸣——在座各位谁没因为点赞这事吵过架？
2. 信息不对等——观众不知道她看到的是谁，悬念拉满
3. "点赞"这个行为有巨大的解读空间——可以是普通互动，也可以是暧昧信号

**剧本还原：**

```
标题：当女朋友看到你给别的女生点赞

0-2s 钩子：
女生刷手机→表情变化→把手机屏幕转向镜头（不给观众看内容）

3-8s 试探期：
女生（平静但阴阳）:"她最近发挺多照片的哈。"
男生（正在打游戏，完全没意识到危险）:"嗯。"
女生（音量微微提高）:"你觉得她好看吗？"
男生（随口）:"还行吧。"
△ 游戏音效突然停了，男生终于感觉不对

9-18s 爆发期：
女生（冷笑）:"还行？我觉得特别好看。尤其是那张自拍。"
男生:"什么自拍？"
女生:"朋友圈第一个啊，你不是昨天刚点过赞吗。"
△ 男生表情凝固，游戏画面显示"GAME OVER"

19-25s 翻盘期：
男生（深吸一口气）:"你等一下。"
△ 男生翻手机→展示屏幕→她昨天的自拍，他点赞了
男生:"你昨天发的照片，我就点了一个赞。"
女生（愣住）:"……那你刚才怎么不说？"
男生:"我就想看看你到底能吃多少醋。"

26-30s 收尾：
女生（恼羞成怒打他）:"你有病啊！"
男生（一边躲一边笑）:"但我喜欢看你吃醋的样子。"
女生（别过头但嘴角上扬）:"滚。"
△ BGM切甜歌，她回头偷看他，他也在偷看她
```

为什么这个本子能火？
1. 情绪曲线完整：紧张→愤怒→错愕→甜蜜，四步走
2. 信息差制造全程悬念：观众和女生一起怀疑，一起被打脸
3. "我就想看看你能吃多少醋"——这句话是全剧之眼，把冲突瞬间转化为糖
4. 收尾的"互相偷看"——糖撒得恰到好处，多一分腻，少一分不够

### 案例二：直男翻车经典结构

**底层逻辑：**

直男梗为什么永远好使？因为它是**双向共鸣**——女生觉得"我男朋友也这样"，男生觉得"我就是这样"。一段对话同时戳中两拨人。

**结构模板：**

```
她说了一句话（字面意思）
他理解成字面意思（真的只理解了字面意思）
她生气了（因为他没理解背后的意思）
他更困惑了（明明按你说的做了为什么还生气）
结局一：他恍然大悟（但已经晚了）
结局二：他做了一件完全跑偏但意外暖心的事
```

## 第九部分：质量检查清单

写完之后，逐条过：

**剧情层面：**
- [ ] 能用一句话说清这条视频讲什么
- [ ] 前3秒有明确的冲突/悬念/反差
- [ ] 每5秒有一个新的信息点
- [ ] 中间至少有一个情绪变化（开心→疑惑、生气→感动等）
- [ ] 结尾有反转或callback或情感落点
- [ ] 没有多余的铺垫和废话

**对话层面：**
- [ ] 每句台词不超过15字
- [ ] 没有"我觉得""我认为""你知道吗"开头
- [ ] 有至少一处"说半句让对方接"
- [ ] 有至少一处用动作代替语言
- [ ] 吵架段落的对话短促（5字内）

**人设层面：**
- [ ] 两个人设性格有明显差异
- [ ] 每个人设至少有一个反差萌点
- [ ] 对话符合各自的人设（不像同一个人写的）

**传播层面：**
- [ ] 标题让人想点进来
- [ ] 前3秒让人想往下看
- [ ] 最后3秒让人想点赞/评论/转发
- [ ] 评论区能产生讨论（不是"好甜"这种无效评论）

**真实感层面：**
- [ ] 场景是日常的（不是搭的景）
- [ ] 对话有语气词和口癖
- [ ] 没有"教材式"的完美对话
- [ ] 情绪有变化过程，不是一步到位
- [ ] 有人在说完最后一句台词后做了一个多余的小动作（最真实的一笔）
