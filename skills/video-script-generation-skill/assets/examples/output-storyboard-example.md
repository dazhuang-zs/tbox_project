# 桂花咖啡：秋日清晨的治愈仪式

## 主要角色
- **角色身份**: 都市轻熟女，咖啡爱好者
- **人物特质**: 干练而不失温柔，追求生活仪式感
- **角色画像**: 28岁职场女性，每天早上都会去咖啡店，享受独处时光

## 故事线
1. **第一幕-日常**：清晨闹钟响起，新的一天开始
2. **第二幕-相遇**：走进咖啡店，桂花香气扑面而来
3. **第三幕-治愈**：品尝桂花咖啡，找到内心的宁静

## 结构框架

### 开场钩子 (第0-3秒)
- **内容**: 特写镜头 - 金色桂花缓缓落入咖啡奶泡，配画外音：
  > "秋天的第一杯治愈"
- **公式**: 视觉美感 + 情感共鸣公式

## 剧本综述 (Script Overview)
核心主题： 在忙碌的都市生活中，用一杯桂花咖啡找到属于自己的治愈时刻
情感基调： 温馨治愈、小确幸
目标受众： 25-35岁都市白领，追求生活品质
总时长： 60秒
