# CSDN 高质量文章写作技能

基于对CSDN平台高访问量文章共性分析，总结出可复用的文章写作框架。核心原则：**热点锚定 + 深度干货 + 反AI味 + 结构利器**。

## 写作前必读

读取参考文档获取完整方法论：
```
references/csdn-writing-guide.md
```

该文件包含：
- 标题公式与10大黄金法则
- 内容结构模板（开篇钩子→目录→正文→金句→结尾）
- CSDN平台流量规则与算法偏好
- 去AI味的8个具体技巧
- SEO优化清单
- 自检清单

## 核心流程

### 1. 选题 — 热点+痛点+价值

CSDN头部文章的核心公式：

```
热点追踪（36小时内最新动态）
  × 痛点切入（评论区的真实问题）
  × 价值输出（可执行的方案/代码/数据）
```

选题优先级：**AI大模型 > 编程实战 > 架构设计 > 面试攻略 > 工具教程**

### 2. 标题 — 不用数字法和疑问法就别发了

CSDN高打开率标题类型优先级：
1. **数字法**：`5个技巧` `3步搞定` `万字长文`
2. **痛点直击**：直接回应用户搜索意图
3. **热点借势**：紧跟最新技术发布
4. **反常识**：颠覆常规认知

禁止使用的标题开头：`你知道吗` `很多人不知道` `今天我要分享`

### 3. 结构 — 黄金三段式

```
开篇（10%）= 痛点引入 + 数据冲击/反常识/个人故事
正文（80%）= 分点展开 + 代码实例 + 数据支撑
结尾（10%）= 核心结论 + 延伸思考 + 互动引导
```

长篇（>3000字）必须配目录导航。

### 4. 正文 — 深度+实战

- 每个观点必须配具体案例或代码
- 数据引用标注来源
- 避免"官方文档搬运工"——要有自己的分析视角
- 代码块有注释，可直接运行
- 图表加分（对比表格、流程图）

### 5. 去AI味 — 这是CSDN读者的核心敏感点

8个必做：
1. 替换AI标志词：“在当今时代”→“2026年”；“总而言之”→删除
2. 每200字至少一句人类独有表达（个人经历、主观判断）
3. 加入批判性看法——AI只会顺着说，你要会反对
4. 制造写作节奏感：长短句交替，避免排比句堆砌
5. 用具体数字替代模糊描述
6. 适当加入口语化表达（“说白了”“这就很扯了”）
7. 引用中文社区的真实梗或事件
8. 结尾不要用“让我们一起期待”之类套话

### 6. SEO与发布

- 关键词出现在：标题、第一段、第一个H2、最后一段
- 标签3-5个，首标签为一级关键词
- 摘要100-150字，含核心关键词
- 发布前用CSDN质量分工具自查

## 文章类型速查

| 类型 | 字数 | 开篇钩子 | 核心要求 |
|------|------|----------|----------|
| 热点快评 | 1500-2500 | 事件+冲击力数据 | 速度快（36h内），观点犀利 |
| 深度干货 | 3000-6000 | 痛点+场景描述 | 代码可运行，案例完整 |
| 趋势分析 | 2500-4500 | 数据对比+判断 | 多维度对比，有独立判断 |
| 实战教程 | 2000-4000 | 目标+前置条件 | 步骤清晰，截图丰富 |
| 系列文章 | 每篇3000+ | 前篇回顾+本篇预告 | 结构化，有连续阅读价值 |
